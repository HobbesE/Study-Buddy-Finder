About Hackbright Academy


Hackbright Academy is a coding bootcamp celebrated for empowering gender diverse programmers to change the gender ratio in technology. Hackbright offers a five week python course for those interested in getting their feet wet in programming, along with an intensive full-stack software engineering bootcamp that provides students with web development skillsets to create web apps like this one... and then some!
